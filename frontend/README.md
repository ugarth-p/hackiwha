# Frontend - User Interface

This is the presentation layer, built with React, focusing on a clean, minimal dashboard to view market insights and branding strategies.

## Tech Stack
- React (Next.js / Vite)
- UI Components: Shadcn UI, TailwindCSS
- State/Data Fetching: React Query / Axios

## Key Features
- **Dashboard:** High-level view of tracked competitors and market sentiment.
- **Alerts Hub:** Real-time notifications for competitor missteps (e.g., "corporate cringe" detected).
- **Strategy View:** Actionable recommendations generated by the AI workers.

## Setup & Run

1. Install dependencies:
   npm install

2. Copy the environment variables:
   cp .env.example .env

3. Start the development server:
   npm run dev
